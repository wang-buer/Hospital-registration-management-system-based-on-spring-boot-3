# 基于springboot的医院挂号管理系统3

## 1、项目介绍

基于springboot的医院挂号管理系统3拥有三种角色

管理员：医生的增删改、患者的增删改、资讯管理

医生：查看待诊患者、病例管理、发送结果报告、留言查看等

患者：预约挂号、查看检查记录、留言等


## 2、项目技术

后端框架：springboot

前端技术：html、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：不需要
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven版本：无限制
- 硬件环境：Windows 或者 Mac OS
